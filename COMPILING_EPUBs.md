# Building YDKJS EPUB Files

```shell
# Install the Pandoc binary...
brew install pandoc

# Create the EPUB file...
# Run the following shell script located in the YDKJS root directory...
./build_epub.sh

# Visit the /epubs directory for compiled EPUB files...
```
